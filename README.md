# 🎬 XUDOMOVIE - Premium Streaming & Movie Database Interface

![XUDOMOVIE Hero](https://via.placeholder.com/1200x400/e50914/ffffff?text=XUDOMOVIE+-+Streaming+Platform)

**XUDOMOVIE** is a modern, responsive, and feature-rich front-end interface for an online streaming and movie database platform. Built with a focus on premium User Experience (UX) and seamless User Interface (UI), it mimics the look and feel of top-tier professional streaming services.

## ✨ Key Features

* **🔍 Advanced Multi-Search & Filtering:**
  * Search for movies, TV shows, and even specific actors in a single smart search bar.
  * Filter browse results dynamically by **Release Year**, **Country of Origin**, and **Genres**.
* **🧠 Smart "Continue Watching":**
  * Automatically saves user watch history to local storage.
  * For TV Shows, it intelligently remembers the exact **Season and Episode** the user last watched.
* **🎞️ Dynamic Trailer Modal:**
  * Watch official YouTube trailers in a sleek, floating pop-up modal without leaving the page.
* **📺 Multi-Server Streaming Integration:**
  * Built-in support for seamlessly switching between multiple third-party video streaming servers.
* **📱 Fully Responsive & Modern UI:**
  * Elegant fading gradients on scrollable tabs.
  * Custom, modernized input fields removing clunky native browser spinners.
  * Fully optimized for Desktop, Tablet, and Mobile devices.

## 🛠️ Tech Stack

This project is entirely front-end and relies on vanilla web technologies, making it incredibly fast and lightweight.

* **HTML5:** Semantic and structured markup.
* **CSS3:** Custom styling, flexbox/grid layouts, and smooth transition animations (No external CSS frameworks like Bootstrap/Tailwind).
* **JavaScript (Vanilla/ES6+):** Handling DOM manipulation, local storage, and async API requests.
* **TMDB API (The Movie Database):** Powering the backend data for movie details, cast, trailers, and advanced search endpoints.

## 🚀 How to Run Locally

Because this is a vanilla Front-End application, setting it up is extremely easy!

1. **Clone or Download** this repository to your local machine.
2. **Get a TMDB API Key:**
   * Go to [TMDB (The Movie Database)](https://www.themoviedb.org/).
   * Create a free account and request an API Key.
3. **Configure API Key:**
   * Open `js/main.js`.
   * Locate the `CONFIG` object at the top of the file and replace the default `API_KEY` with your own key.
4. **Launch the App:**
   * You can simply open `index.html` directly in your web browser.
   * *Recommended:* Use a local server like the **Live Server** extension in VS Code for the best experience and to avoid CORS issues.
